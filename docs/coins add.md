অ্যাডমিন হিসেবে গিফট কোড তৈরি করার জন্য আপনার একটি আলাদা **"Admin Panel"** বা টুল প্রয়োজন হবে। যেহেতু সাধারণ ইউজাররা তাদের ব্যালেন্স থেকে গিফট দেয়, কিন্তু আপনি অ্যাডমিন হিসেবে **ফ্রি-তে (ব্যালেন্স ছাড়া)** কোড তৈরি করবেন।

এর জন্য আমি আপনাকে একটি **`admin.html`** ফাইল তৈরি করে দিচ্ছি। এই ফাইলটি আপনি আপনার ওয়েবসাইটে আপলোড করবেন না (অথবা পাসওয়ার্ড দিয়ে প্রটেক্ট করে রাখবেন), অথবা শুধু আপনার কম্পিউটার থেকে লোাললি ওপেন করে ব্যবহার করবেন।

### এই `admin.html` কিভাবে ব্যবহার করবেন?

১. ইউজার যখন টেলিগ্রামে আপনাকে মেসেজ দিয়ে পেমেন্ট কনফার্ম করবে।
২. আপনি আপনার কম্পিউটার বা মোবাইল থেকে `admin.html` ফাইলটি ওপেন করবেন।
৩. **Amount** ঘরে কত কয়েন দিতে চান তা লিখবেন (যেমন: 1000)।
৪. **Generate Code** বাটনে চাপ দেবেন।
৫. একটি কোড তৈরি হবে (যেমন: `BUY-X7Z...`)। সেটি কপি করে টেলিগ্রামে ইউজারকে পাঠিয়ে দেবেন।
৬. ইউজার সেই কোডটি `redeem.html` পেজের "Receive Gift" অপশনে বসালে তার ব্যালেন্স অ্যাড হয়ে যাবে।

### পদ্ধতি ২: সরাসরি Firebase Console থেকে (বিকল্প পদ্ধতি)

আপনি যদি আলাদা ফাইল বানাতে না চান, তবে সরাসরি Firebase ডাটাবেস থেকেও কোড তৈরি করতে পারেন:

1. [Firebase Console](https://console.firebase.google.com/)-এ যান।
2. **Realtime Database**-এ ক্লিক করুন।
3. `gifts` লেখার পাশে **+ (প্লাস)** চিহ্নে ক্লিক করুন।
4. **Name** এর ঘরে একটি কোড দিন (যেমন: `SPECIAL-100`)।
5. **Value** এর ঘরে নিচের মতো JSON ফরম্যাটটি ম্যানুয়ালি লিখুন:
   ```json
   {
     "amount": 100,
     "status": "active",
     "type": "admin"
   }
   ```
6. **Add** বাটনে ক্লিক করুন।
7. এখন ইউজারকে `SPECIAL-100` কোডটি দিলে সে ১০০ কয়েন পাবে।

### ⚠️ নিরাপত্তা সতর্কতা
আমি যে `admin.html` ফাইলটি দিয়েছি, সেটি **পাবলিকলি শেয়ার করবেন না**। কারণ এই ফাইলে কোনো লগইন সিস্টেম নেই। যে কেউ এই ফাইলটি ওপেন করলেই আনলিমিটেড কয়েন জেনারেট করতে পারবে।
*   এটি শুধু আপনার নিজের কাছে (ল্যাপটপ বা মোবাইলে) অফলাইনে সেভ করে রাখুন।
*   প্রয়োজন হলে ফাইলটি ওপেন করে কোড বানিয়ে আবার বন্ধ করে দিন।
